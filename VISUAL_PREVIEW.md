## Interaction Flow

### Starting a Timer
```
1. Tap Timer tab
   └→ Timer screen opens (00:00:00)

2. Tap START button
   └→ Timer begins counting
   └→ Button changes to PAUSE
   └→ LAP button appears

3. Tap LAP button (optional)
   └→ Current time recorded
   └→ Added to lap list

4. Tap PAUSE button
   └→ Timer stops
   └→ Button changes to START
   └→ RESET button appears

5. Tap RESET button
   └→ Timer returns to 00:00:00
   └→ Lap list clears
```

### Setting a Countdown
```
1. Tap Countdown tab
   └→ Countdown screen opens with time picker

2. Set time using +/- buttons
   └→ Hours: 0-23
   └→ Minutes: 0-59
   └→ Seconds: 0-59

3. Tap START COUNTDOWN
   └→ Countdown begins
   └→ Large display shows remaining time
   └→ PAUSE/RESUME and RESET buttons appear

4. When countdown reaches 0
   └→ "Time's Up!" message appears
   └→ Display turns red
   └→ RESET button available
```

### Adding an Alarm
```
1. Tap Alarm tab
   └→ Alarm screen opens

2. Tap + (FAB) button
   └→ Add Alarm dialog opens

3. Select time using time picker
   └→ Scroll or type hours/minutes

4. Add label (optional)
   └→ Type descriptive text

5. Tap SAVE
   └→ Dialog closes
   └→ New alarm appears in list
   └→ Alarm is enabled by default
```
